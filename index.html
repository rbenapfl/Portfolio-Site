  • Rose Colored Comments!

    Quick and dirty Google Chrome extension I whipped up at a DBC hackathon with one other guy (talk about being a seasoned pair programmer). We coded a Chrome extension that lets you view youtube comments in a window next to the video player without scrolling down as well as providing a word filtering feature to remove all comments that have a negative flavor. Neither of us had knowledge about how to code a chrome extension beforehand yet we made one that interacts with the browser window as well as utilizing the Youtube API within a few hours!

  • Using this extension

    You can run this extension by simply downloading, unzipping, and going to your extensions in Chrome. Enable Developer Mode, hit Load Unpacked Extension, and select the Source Folder. The extension fires up on click so head over to your favorite youtube video and click the extension icon for some positive comment viewing (bonus points if you customize the word filter list to your liking in the JS file). View the Github Repo and some pictures of it in action!
